戚风蛋糕

配料表

  • 鸡蛋5个
  • 牛奶40g
  • 油40g
  • 面粉85g
  • 糖30g
  • 盐2g
  • 糖60g
  • 柠檬汁5ml

制作步骤

  1. 鸡蛋蛋白蛋黄分离
  2. 蛋黄糊：蛋黄+前5个配料
  3. 蛋白糊：蛋白打发+后2个配料
  4. 烤箱预热5分钟，150℃ 双层 40分钟

冰淇淋

配料表

  • 淡奶油200ml (450g)
  • 鸡蛋黄2个 (3个)
  • 纯牛奶50ml (130g)
  • 白糖50g (70g)
  • 盐少许 (1g)
  • 柠檬汁少许

制作步骤

  1. 蛋黄+糖+柠檬汁/醋 搅拌 至 浅黄色
  2. 牛奶加热 冒泡泡 + 蛋黄液 搅拌
  3. 蛋黄液+盐，小火快速搅拌至沸腾
  4. 打发淡奶油至9分发，加入蛋奶液

爆米花

配料表

  • 玉米粒100g
  • 油40-50g
  • 黄油20-30g
  • 水40g
  • 砂糖80g

制作步骤

  1. 大火 + 玉米 + 油 爆开
  2. 糖 + 水 融化 中大火 → 黄
  3. +黄油 → 焦糖色
  4. +已爆米花 搅拌

奶黄馅

配料表

  • 鸡蛋100g (2个)
  • 棉白糖45g
  • 澄粉/小麦淀粉25g
  • 奶粉15g
  • 纯牛奶80g
  • 无盐黄油30g

制作步骤

  1. 鸡蛋打散 + 牛奶 + 奶粉 + 黄油
  2. 小火翻拌炒
  3. 冷藏
  4. 分成25g/个的小份

蛋黄流沙

配料表

  • 咸蛋黄3个
  • 鸡蛋黄1个
  • 奶粉20g
  • 吉士粉3g
  • 绵白糖40g
  • 动物黄油35g

制作步骤

  1. 蛋黄以外的材料揉捏
  2. + 粉状蛋黄
  3. 揉捏至软 → 冰箱

注：2种蛋黄均需要压成粉状。
